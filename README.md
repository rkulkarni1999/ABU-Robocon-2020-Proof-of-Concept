# ABU Robocon 2020 Proof of Concept. 

This project outlines the design and development process of the Robots, developed by Team Vector for DD-ROBOCON 2020. It covers various aspects of the Robots including Mechanisms, CAD Designs, Calculations, Dimensions and the Electronics subsystem. The main objective of the team was to satisfy and accomplish various tasks as-per the gameplay while meeting the ABUâ€™s rules and regulations with special consideration given to Safety.

The project contains the developmental aspects of the two robots developed by Team Vector that will compete at DD
ROBOCON 2020, India.
It is subdivided in 3 sections i.e.
Pass Robot (PR), Try Robot (TR) and Electronic
Subsystems. Every subsection a brief idea about the Base Drives of
PR and TR, Ball Picking and Passing mechanism, Ball
receiving mechanism, Try Mechanism and Kicking
Mechanism.
